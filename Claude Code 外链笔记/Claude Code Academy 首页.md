# Claude Code Academy 首页

- 来源：https://academy.claude-code.club
- 抓取时间：2026-03-17
- 类型：中文学习站点首页

## 核心内容

这是一个面向中文开发者的 Claude Code 学习导航站，首页把内容分成四层：

- 入门指南：概念、API Key、安装配置、套餐计费、快速体验、常见错误和 FAQ。
- 实战技巧：通用资源、工作流程、UI 设计、命令技巧、Subagent、MCP Tools。
- 完整课程：系统化课程和视频。
- 项目案例：真实案例拆解。

## 我提炼的重点

- 这不是一篇教程，而是中文资源聚合入口。
- 它最大的价值是帮你按学习阶段找材料，而不是给你最终答案。
- 如果你更偏中文资料、社区案例和系统课程，这个站点比单篇文章更方便回查。

## 怎么用最划算

- 入门阶段：从首页的“入门指南”走一遍。
- 提升阶段：重点进入“实战技巧”和“完整课程”。
- 缺案例时：看“项目案例”，把学习内容和真实交付连接起来。
