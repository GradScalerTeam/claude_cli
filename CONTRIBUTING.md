# 贡献指南

感谢你对 Claude CLI 项目的关注！

## 如何贡献

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 开发指南

### 运行测试

```bash
# 安装依赖
npm install

# 运行测试
npm test

# 运行特定测试
npm test -- tests/structure.bats
```

### 代码风格

- 使用 2 空格缩进
- 遵循现有代码风格
- 添加测试覆盖新功能

## 报告问题

请在 GitHub Issues 中报告问题。
